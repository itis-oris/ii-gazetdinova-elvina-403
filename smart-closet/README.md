# Smart Closet

Smart Closet — это веб-приложение для управления личным гардеробом.  
Пользователь может добавлять вещи, вести вишлист, сохранять покупки и получать рекомендации по сочетанию цветов.

## Функциональность

### 1. Управление вещами
- добавление вещи;
- редактирование вещи;
- удаление вещи;
- просмотр списка всех вещей пользователя.

### 2. Вишлист
- добавление вещей в список желаемых покупок;
- просмотр вишлиста.

### 3. Покупки
- добавление покупки;
- просмотр списка покупок пользователя;
- сохранение даты покупки, цены и заметки.

### 4. Цветовые рекомендации
- при сохранении вещи приложение обращается к внешнему API цветов;
- автоматически определяется название цвета;
- вычисляется дополняющий цвет;
- на основе дополняющего цвета можно подбирать подходящие вещи.

### 5. Авторизация
- регистрация пользователя;
- вход в систему;
- работа с личными данными только авторизованного пользователя.

## Технологии

- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- PostgreSQL
- FreeMarker
- HTML / CSS / JavaScript
- Maven

## Структура проекта

- `controller` — контроллеры приложения;
- `service` — бизнес-логика;
- `repository` — работа с базой данных;
- `entity` — сущности JPA;
- `templates` — FreeMarker-шаблоны;
- `static` — CSS и JavaScript.

## Основные сущности

- `User` — пользователь;
- `Item` — вещь;
- `WishlistItem` — элемент вишлиста;
- `Purchase` — покупка;
- `Category` — категория;
- `Brand` — бренд;
- `Tag` — тег.

## Запуск проекта

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/elvinka505/smart-closet.git
   ```

2. Перейти в папку проекта:
   ```bash
   cd smart-closet
   ```

3. Настроить `application.properties`:
- указать параметры подключения к PostgreSQL;
- указать настройки базы данных;
- при необходимости настроить внешний API.

4. Запустить приложение:
   ```bash
   ./mvnw spring-boot:run
   ```

5. Открыть в браузере:
   ```text
   http://localhost:8080
   ```

## Идея проекта

Проект создан как приложение для удобного хранения информации о вещах и планирования гардероба.  
Smart Closet помогает не только хранить список одежды и аксессуаров, но и отслеживать покупки, а также использовать рекомендации по цветовым сочетаниям.
